智能广告机

发表于 2017-05-05   |  

###简介
本项目基于Face++官方网址,根据官方提供的开发文档完成的人脸识别功能，api能够识别一张人像图片，并返回人物的年龄、性别、微笑指数等数据信息，系统根据三个指标匹配出对应的广告数据，点击广告数据可查看详情信息

###环境搭建
AndroidStudio2.3 + compileSdkVersion 25 + build:gradle:2.2.3

###主要类及功能
LoadingActivity：登录Face++,识别授权状态，用户需配置API_KEY，API_SECRET
MainActivity：上传图片，识别，拍照识别，管理员登录，管理广告分类，添加广告
FaceIdentifyActivity：图像识别类，根据识别的结果进行广告展示

PLSQL安装配置

发表于 2017-05-05   |  

###PLSQL64位安装完后，依赖数据库需要做配置
基本配置：Tools->Preferences->Oracle Home,OCI(Oracle Call Interface) library
因为PLSQL需配置对应的数据库配置，此处配置可不安装oracle客户端。具体配置如下 

Oracle Home：磁盘路径:\instantClient  
OCI library：磁盘路径:\instantClient\oci.dll

遇到的问题，需配置ODBC数据源管理程序
cmd运行命令，odbcad32 打开ODBC数据源管理程序
添加一个excellocal 32位 driver do Microsoft Excel(.xls)，配置完后即可import excel文件

###利用PLSQL导入excel数据
导入cvs格式的数据到oracle，Data from Textfile
Tools->Text Importer->Open data file
选择对应的cvs文件，添加进来后，选择Data to Oracle oracle数据库应该创建好对应的数据库表，且表结构和cvs表头属性字段一一对应随后Import即可

AndroidStuio每次升级带来的烦恼

发表于 2016-03-27   |  

近日又按照提示，升级了预览版的AS，在做小白鼠的途中，被升级所带来的各种插件丢失
所扰。于是决定对AS在mac中的存储目录结构一探究竟。
首先利用AppCleaner搜索所有与AS有关的文件，最后查到，两大类目录

这两个目录里AS竟然保存了每个版本，从低到高都保存着。这严重影响了使用，可进行
低版本删除工作，不影响高版本工作。
最最头疼的是跟新后插件都不见了，这个升级机制太蛋疼了，于是查看各个文件夹，
最终在Application Support找到了历史记录，把低版本的Support copy一份给最
新版本即可，不用重复下载。
这体验还望google早日优化啊!
