PLSQL安装配置

发表于 2017-05-05   |  

###PLSQL64位安装完后，依赖数据库需要做配置
基本配置：Tools->Preferences->Oracle Home,OCI(Oracle Call Interface) library
因为PLSQL需配置对应的数据库配置，此处配置可不安装oracle客户端。具体配置如下 

Oracle Home：磁盘路径:\instantClient  
OCI library：磁盘路径:\instantClient\oci.dll

遇到的问题，需配置ODBC数据源管理程序
cmd运行命令，odbcad32 打开ODBC数据源管理程序
添加一个excellocal 32位 driver do Microsoft Excel(.xls)，配置完后即可import excel文件

###利用PLSQL导入excel数据
导入cvs格式的数据到oracle，Data from Textfile
Tools->Text Importer->Open data file
选择对应的cvs文件，添加进来后，选择Data to Oracle oracle数据库应该创建好对应的数据库表，且表结构和cvs表头属性字段一一对应随后Import即可
